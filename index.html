Значение символов в грамматике

Заглавная латинская буква - рисование ветки
Строная латинская буква - служебный символ для построения грамматик
+ - поворот против часовой стрелки
- - поворот по часовой стрелке
[ - сохранить начальную позицию в стек
] - взять из стека сохраненную позицию

Стреловидная кривая Серпинсеого
